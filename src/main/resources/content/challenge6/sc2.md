### What's the deal?
The owner is considering making a family discount for families that are all members of the club, however she wants to know how many families are part of the club beforehand. She counts a family to be a group of 3 or more people with the same last name. What is the number of families in this list of names?
::: hint
Use a dictionary to keep track of surnames and the number of occurrences. Loop through the list of names, and add one to the stored value for each time a name appears. Then go through the dictionary and see which surnames have 3 or more occurences.
:::
